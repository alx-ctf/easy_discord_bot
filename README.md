# easy_discord_bot
очень простой дс бот + гайд по его созданию

структура бота 
```py
easy_discord_bot/
├── .gitignore                 # чтобы гит не записывал ненужные файлы 
├── requirements.txt           # список зависимостей
├── .env.example               # файлик созданный для этого проекта для демонстрации
├── README.md                  
├── bot.py                     # основа бота
├── core/                   
│   ├── __init__.py         
│   ├── localization.py        # основной модуль локализации
│   └── strings/            
│       ├── en.json            # английский язык
│       └── ru.json            # русский язык
├── cogs/
│   └── ping.py                # команда (cog) с поддержкой языка
└── data/
    └── guilds.json            # простое хранилище настроек серверов (язык)
```
